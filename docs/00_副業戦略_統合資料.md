## エグゼクティブサマリー

### 結論

**Chrome拡張 × テンプレート販売 × AIメディア** の3層構造で副業を立ち上げる。

| 指標 | 数値 |
|---|---|
| 初期投資 | ¥10,000以下 |
| 損益分岐 | **Month 2** |
| Year 1 年間利益（標準） | **¥973,422** |
| Month 12 単月利益 | **¥177,605** |
| 年間稼働 | 約360時間（週5-10h） |
| 時間効率 | **¥2,704/時間** |
| 利益率 | **87%** |

### なぜこの戦略か

11種類のビジネスモデルを検討した結果、以下の理由で3層フライホイールを採用：

1. **¥/時間が最も高い**：Chrome拡張（¥6,250〜25,000/h）> アフィリエイト（¥500〜2,500/h）
2. **各層が相互に育て合う**：拡張ユーザーの声→記事ネタ、メディアSEO→拡張DL
3. **単一障害点がない**：3つの収益源が独立
4. **仕組み化できる**：12ヶ月目に週4時間で月16〜45万円

- **AIYouTube**：収益化（登録者1,000+、再生4,000h+）に6-12ヶ月

## 採用戦略：3層フライホイール

```
Layer 1（即金）    Layer 2（集客）    Layer 3（資産）
Chrome拡張  ←→  AIメディア   →  SaaS化/売却
テンプレート ←→  SEO流入     →  月額課金
```

### Layer 1：即金エンジン（Week 1-4）

**Chrome拡張「PagePulse - AI Website Analyzer」** ✅ MVP完了
- **ニッチ：** Webページ総合分析（SEO/速度/A11y/構造）
- **言語：** 英語ファースト（グローバル市場狙い）
- コア機能（無料）：1クリックでSEO・パフォーマンス・アクセシビリティ・構造分析
- プレミアム（¥1,480/月）：AI提案・CRO診断・EC特化分析
- 課金基盤：ExtensionPay（Stripe連携）
- **リポジトリ：** https://github.com/knomoto-comrade/pagepulse

**テンプレート3種**（未着手）
- Looker Studio EC分析ダッシュボード（¥9,800）
- GTM設計書テンプレート Notion版（¥4,980）
- GA4計測チェックリスト PDF（¥2,980）

### Layer 2：集客エンジン（Week 5-12）

**AI特化メディア**
- ニッチ：GA4/GTM/EC分析 × CRO
- AIパイプライン：Claude Haiku → 人間追加 → WordPress自動投稿
- 月12記事を週2.5時間で運用

### Layer 3：資産エンジン（Month 4+）

- Chrome拡張の検証済み機能をWeb SaaS化
- or 事業として売却（月収の24-36倍が相場）
