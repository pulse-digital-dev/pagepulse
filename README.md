# PagePulse - AI Website Analyzer

A Chrome extension that analyzes any webpage instantly, providing actionable SEO, performance, and accessibility insights with one click.

## Features

- **SEO Analysis**: Title, meta description, H1, canonical, OGP, heading hierarchy
- **Performance**: Image dimensions, lazy loading, render-blocking scripts
- **Accessibility**: Alt text, lang attribute, form labels, viewport meta
- **Structure**: JSON-LD structured data, robots meta, favicon, HTTPS, page size

## Installation (Developer Mode)

1. Clone this repository
2. Open `chrome://extensions/` in Chrome
3. Enable "Developer mode"
4. Click "Load unpacked" and select the project folder
5. Navigate to any website and click the PagePulse icon

## Tech Stack

- Chrome Extension Manifest V3
- Vanilla JavaScript (no dependencies)
- CSS with custom properties and glassmorphism design
