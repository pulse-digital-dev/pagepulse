# 副業ビジネス立ち上げワークフロー

## 前提

- side-business-strategy スキルを適用
- 主軸：Chrome拡張（PagePulse） × テンプレート販売 × AIメディア の3層フライホイール
- **進捗：** Week 1 完了（2026-03-23）、Week 2 着手中
- 稼働上限：週10時間（構築期）、週5時間（運用期）

---

## Phase 1：即金エンジン構築（Week 1-4）

### Week 1：Chrome拡張 MVP開発 ✅ 完了

1. ✅ Chrome拡張のアイデアを決定する → **PagePulse（AI Website Analyzer）** を選定
   - ニッチ選定で5人の専門家会議 + Why×3ブラッシュアップ実施
   - 英語ファースト・グローバル市場戦略を決定
2. ✅ Manifest V3でプロジェクトを初期化する
3. ✅ コア機能を実装する（SEO/パフォーマンス/A11y/構造の4領域分析）
4. ✅ UIレンダリング検証完了
5. ✅ GitHubリポジトリ作成・push（knomoto-comrade/pagepulse）
6. [ ] ユーザーによるChromeへの手動ロード・実機テスト

### Week 2：公開準備 + SNS開設 ← 現在地

1. [ ] Chrome Web Storeにデベロッパー登録する（$5）
2. [ ] ストアの説明文・スクリーンショットを作成する
3. [ ] Chrome Web Storeに申請する（審査1-3日）
4. [ ] X（Twitter）匿名アカウントを開設する（ペンネーム + アイコン）
5. [ ] Product Huntにリスティングを作成する

### Week 3：テンプレート3種作成 + 出品

1. Gumroadアカウントを開設する
2. テンプレートを3種作成する：
   - Looker Studio EC分析ダッシュボード（¥9,800）
   - GTM設計書テンプレート Notion版（¥4,980）
   - GA4 eコマース計測チェックリスト PDF（¥2,980）
3. Gumroadに出品 + GA4連携設定する
4. note にもクロスリスティングする

### Week 4：初期マーケティング

1. Xで無料テンプレート（1種）を配布投稿する
2. Qiita/Zennに技術記事を1本書く（Chrome拡張紹介）
3. 拡張のユーザーフィードバックを収集する
4. **KPI確認**：拡張ユーザー数・テンプレ販売数を記録

## Phase 2：課金導入 + メディア基盤（Week 5-8）

### Week 5：フリーミアム課金導入

1. ExtensionPayを導入してStripe連携する
2. 無料版/プレミアム版の機能分割を実装する
3. Chrome Web Storeを更新する
4. **KPI確認**：初課金ユーザーの有無

### Week 6：メディア基盤構築

1. ドメインを取得する（お名前.com等）
2. WordPress + テーマを設定する
3. GA4 / GTM計測設計を行う
4. バーチャルオフィスを契約する（特定商取引法対策）

### Week 7-8：AI記事パイプライン構築

1. Claude Haiku APIキーを取得する
2. GASで記事下書き自動生成スクリプトを構築する
3. WordPress REST API連携を設定する
4. 初期10記事を生成＋専門知識追加＋投稿する
5. ASP登録する（A8.net / もしもアフィリエイト）

---

## Phase 3：最適化 + Go/No-Go判定（Week 9-12）

### Week 9-10：収益導線完成

1. メディア記事内にアフィリエイトリンクを設置する
2. Chrome拡張→メディアの導線を設置する
3. メディア→テンプレート販売CTAを設置する
4. CROで導線を最適化する

### Week 11：プロダクト改善

1. Chrome拡張の機能追加（ユーザー要望ベース）
2. v1.1としてリリースする
3. テンプレートの追加作成（売れ筋ジャンル）

### Week 12：Go/No-Go判定

1. 以下のKPIを計測する：

| KPI | Go基準 | No-Go基準 |
|---|---|---|
| Chrome拡張ユーザー数 | 500+ | 100未満 |
| フリーミアム課金者 | 10人以上 | 3人未満 |
| テンプレート販売数 | 月10件以上 | 月3件未満 |
| メディアPV | 月500+ | 月100未満 |
| 合計月収 | ¥30,000以上 | ¥5,000未満 |

2. **Go** → Phase 4（スケール）に進む
3. **No-Go** → Chrome拡張のアイデアを変えて Week 1 からスプリント再実行

---

## Phase 4：仕組み化 + スケール（Month 4以降）

1. 仕組み化チェックリスト（スキル参照）を全項目クリアする
2. 売上レポートの自動生成を設定する（Looker Studio）
3. 月間稼働を20時間以下に最適化する
4. Layer 3（SaaS化 or 事業売却）を検討する
